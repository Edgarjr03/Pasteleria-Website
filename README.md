# Pastelería Website 

## Español
Sitio web completo para una pastelería artesanal desarrollado con HTML, CSS y JavaScript puro.
Permite a los clientes explorar el menú, personalizar su pedido y enviarlo directamente 
por WhatsApp. Incluye un panel de administración para gestionar productos y configuración 
del negocio.

### Funcionalidades
- Catálogo de postres con imágenes reales
- Configurador de pedidos personalizado por sabores y combinaciones
- Envío de pedido directo por WhatsApp
- Panel de administración para gestionar productos
- Diseño responsivo y elegante

### Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts

### Cómo ejecutar
1. Clona el repositorio
2. Abre `Index.html` en tu navegador
3. Para el panel admin abre `IndexAdmin.html`
4. Nota: el adminpanel solo funciona para hacer cambios localmente lo utilizo para pruebas

## English
Complete website for an artisan bakery developed with pure HTML, CSS and JavaScript.
Allows customers to browse the menu, customize their order and send it directly 
via WhatsApp. Includes an admin panel to manage products and business settings.

### Features
- Dessert catalog with real product images
- Custom order configurator by flavors and combinations
- Direct WhatsApp order submission
- Admin panel for product management
- Responsive and elegant design

### Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts

### How to run
1. Clone the repository
2. Open `Index.html` in your browser
3. For the admin panel open `IndexAdmin.html`
4. Note: the admin panel only works for making changes locally, I use it for testing
